Function buildOpenWeatherMapRequest

  • buildOpenWeatherMapRequest(token: string, latitude: number, longitude: number, options?: OpenWeatherMapOptions): OpenWeatherMapBuilder

  • Conveience function for creating a Request Chain.

    Example usage:

    // Only get the hourly forecast, and extend it.
    buildOpenWeatherMapRequest("api-token", 42, -42)
     .units(Units.Metric)
     .onlyHourly()
     .execute()
